# Emacs Packages

## Paquetes recomendados
- `magit`: Git en Emacs
- `projectile`: navegacion de proyectos
- `company`: autocompletado
- `flycheck`: linting
- `which-key`: ayuda de atajos

## Atajos basicos
- `C-x C-f`: abrir archivo
- `C-x C-s`: guardar
- `C-g`: cancelar
- `M-x`: ejecutar comando

## Nota
Si usas `use-package`, puedes habilitar paquetes de forma declarativa.
