# Vim Commands

## Navegacion
- `h j k l`: mover el cursor
- `gg`: inicio del archivo
- `G`: final del archivo
- `:n`: ir a linea n

## Edicion
- `i`: insertar
- `a`: insertar despues
- `o`: nueva linea abajo
- `dd`: borrar linea
- `yy`: copiar linea
- `p`: pegar

## Buscar y reemplazar
- `/texto`: buscar
- `n` / `N`: siguiente / anterior
- `:%s/old/new/g`: reemplazo global

## Guardar y salir
- `:w`: guardar
- `:q`: salir
- `:wq`: guardar y salir
