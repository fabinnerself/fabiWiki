# VS Code Extensions

Lista corta de extensiones y atajos para uso diario.

## Extensiones basicas
- ESLint
- Prettier
- GitLens
- EditorConfig
- Markdown All in One

## Atajos utiles
- `Ctrl+P`: abrir archivo rapido
- `Ctrl+Shift+P`: paleta de comandos
- `Ctrl+K` `Ctrl+S`: ver todos los atajos
- `Alt+Shift+F`: formatear documento

## Configuracion recomendada
```json
{
  "editor.formatOnSave": true,
  "editor.rulers": [80, 100],
  "files.trimTrailingWhitespace": true
}
```
